Les h1 sont aussi des éléments { display: block; }

{ display: block; }
Les element block peuvent avoir:
une hauteur, une largeur, une marge exterieur, une bordure, une marge interne

Pour penser une section de façon responsive, je peux mettre la largeur en pourcentage { width: 90%; } et je peux si je le souhaite aussi lui donner une largeur minimum { min-width: 320px; } et une largeur maximum { max-width: 1080px; }
